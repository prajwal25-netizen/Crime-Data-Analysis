Introduction

Crime data analysis is essential for identifying patterns, trends, and areas requiring law enforcement attention. This report explores a publicly available U.S. crime dataset sourced from Kaggle. The goal is to analyze crime trends over time, examine state-wise distributions, and conduct an in-depth analysis of Los Angeles, California.

By understanding how crimes vary by time, region, and demographics, data analysts can provide valuable insights for crime prevention, policy planning, and public safety improvements.

Dataset Description

The dataset includes nationwide crime reports across several years and states in the U.S. Key features include: - Year: The year the crime was reported - State and City: The location of the crime - Crime Type, Weapon, Crime Solved: Information about the crime event - Victim Age, Victim Sex, Victim Race, Relationship: Victim demographics

The dataset contains thousands of observations, offering sufficient depth for statistical and trend analysis.

Crime Trends Over Time

total_crime <- crimedata %>%
  group_by(Year) %>%
  summarize(count = n())

ggplot(total_crime, aes(x = Year, y = count)) +
  geom_line(color = "steelblue", linewidth= 1) +
  scale_x_continuous(breaks = seq(min(total_crime$Year), max(total_crime$Year), by = 5)) +
  labs(
    x = "Year",
    y = "Total Number of Crimes",
    title = "Crime Trends Over Time")+
      theme_minimal()

Statewise Crime Distribution

statewise <- crimedata %>%
  group_by(State) %>%
  summarize(number = n())

ggplot(statewise, aes(x = reorder(State, -number), y = number)) +

Victim Age Group Distribution

agegroup <- LA %>%
  mutate(agegroup = case_when(
    `Victim Age` < 5 ~ "Under 5",
    `Victim Age` >= 5 & `Victim Age` < 18 ~ "5-17",
    `Victim Age` >= 18 & `Victim Age` < 50 ~ "18-49",
    `Victim Age` >= 50 ~ "50+"
  ))

Conclusion

This analysis uncovered several valuable insights from the U.S. crime dataset:

  • Crimes have shown varying trends over the years, with some periods of increase or decline.
  • Certain states contribute disproportionately to national crime totals.
  • Los Angeles, as a major urban area, shows distinct crime patterns — including common weapons used and age demographics.
  • Linear regression analysis shows time-based trends which could help in forecasting and prevention strategies.

Overall, this type of analysis can support policymakers, safety organizations, and researchers in making data-informed decisions. into how crime patterns vary by time, location, and demographic factors. The linear regression model suggests trends that can be further explored for predictive insights. The city-specific breakdown helps focus policy and safety measures.
